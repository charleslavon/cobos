About this page

This page shows you how to create a master/detail view using Bootcards. Main features:

  • Markup based on Bootstrap 3+.
  • Near-native iOS and Android look (there's one for Desktop too).
  • Fixed header and footer.
  • Independent scrolling columns.
  • Automatic single pane portrait mode (similar to the iOS Mail app): make sure you also view this page using a tablet and rotate to portrait.
  • Off-canvas menu. On this page it is configured to push the main content away, it can also be configured as an overlay.

This is only a sample page: it is not meant to be fully functional. We've used Font Awesome for the icons and have include FTLabs' FastClick library to make tapping links on iOS faster.
